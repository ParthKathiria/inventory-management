# Inventory Management
A Full Stack Inventory Management Dashboard Application using Nextjs, Redux, Node, and AWS.

Following are the sub-pages:

1. Dashboard Page: Contains detailed overview and statistics about the products and sales with visuals like charts and graphs.
2. Inventory Page: Shows the inventory of products
3. Products Page: Provides a visual of the products in the inventory and a functionality to add a new product to the inventory.
4. Users Page: A list of users with some of their info.
5. Settings Page: A typical settings page with the name, email address of the admin user along with a toggle setting to enable/disable notifications and dark mode, and a language setting for the web-app.
6. Expenses Page: Provides a detailed summary about the expenses for a company using a pie chart.
